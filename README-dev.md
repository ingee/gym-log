Gym-log
======
Daily workout log

## What I use
* @vue/cli 4.5.7

## What I did
```
npm install -g @vue/cli
vue create gym-log
  # select "Manually select..."
  # select vue2, babel, router, vuex, lint
  # select no history-mode for router
  # select ESLint + Standard
  # select Lint on save
  # select config in dedicated config files
vue add vuetify
```
